# Automation

Status: Active.

## Jobs

- `RTIH hourly benchmark research`
  - Cadence: every hour.
  - Workspace command: `npm run research:hourly`.
  - Purpose: capture 3-5 additional benchmark sites, append evidence, update status, and log failures.

- `RTIH six-hour design intelligence`
  - Cadence: every 6 hours.
  - Workspace command: `npm run research:report`.
  - Purpose: summarize emerging patterns, strongest layouts, weakest layouts, overused clichés, and visual anti-patterns.

- `RTIH daily strategy refinement`
  - Cadence: every 24 hours.
  - Workspace command: `npm run research:daily`.
  - Purpose: refresh brand, typography, CTA, navigation, storytelling, homepage narrative, and anti-slop guidance from accumulated evidence.

## Constraints

- No frontend implementation code.
- Research recommendations remain provisional until screenshot database, design audit, style guide, UX principles, information architecture, and anti-slop review are complete and approved.
- Any current-site uncertainty must be flagged instead of treated as fact.
