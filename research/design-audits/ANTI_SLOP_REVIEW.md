# Anti-Slop Review

Status: Active gate.

## Forbidden Defaults

- Generic gradients.
- Random floating shapes.
- Fake startup metrics.
- Fake founder quotes.
- Fake ecosystem claims.
- Artificial unicorn positioning.
- Cliché innovation visuals.
- Crypto-style aesthetics.
- Excessive glassmorphism.
- Decoration-led motion.
- Startup buzzword overload.
- Generic stock-photo startup teams.
- Fake diversity imagery.
- Dark mode without strategic need.

## Approval Questions

1. Is this authentic to RTIH?
2. Is this strategically useful?
3. Is this visually timeless?
4. Is this improving clarity?
5. Is this improving credibility?
6. Is this improving founder trust?
7. Is this improving navigation?
8. Is this improving conversion?
9. Is this based on evidence?
10. Is this just trendy nonsense?
11. Would a world-class design director approve this?
12. Would this still look good in 5 years?
13. Is this mobile-first?
14. Is this accessible?
15. Is this scalable?
16. Is this visually coherent?
17. Is this overdesigned?
18. Is this underdesigned?
19. Is this emotionally compelling?
20. Is this institutionally trustworthy?

Any failed answer requires revision.
